# Motion Detector App

This is a Final Project for the Python Programming class b(Academic Year 2020)

It is built with [opencv](https://pypi.org/project/opencv-python/) library.

It is tested on Windows.

Features and Usage:

- The app looks for a moving object. The object size is specified by the program.
- When it detects the motion, it plays a sound.
- To quit the window, Press the key, 'q' on the keyboard
